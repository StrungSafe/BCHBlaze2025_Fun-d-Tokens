# BCHBlaze2025_Fun-d-Tokens

## Minting Your Fund

The minting contract will allow users to mint a nft  w/ proof of collateral sent to the asset redemption contract

## Redeeming Your Fund

To redeem your fund's NFT and retrieve the assets the burn contract and redeem contracts much be run in the same transaction

### Burn Fund NFT

The proof of fund NFT allows the UTXO to be spent when a chosen token is proven to be burned

### Move the Assets

You are allowed redeemption of the fund's token for assets when the fund contracts sees the proof of burn token is included in the transaction
